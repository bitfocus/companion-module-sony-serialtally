## Sony XVS

This module is for controlling Sony XVS video switchers/vision mixers.

### Actions:

-   XPT M/E
-   XPT Aux
-   Auto Transition M/E
-   Key On/Off
-   Recall Snapshot
-   Macro Recall/Take
-   Activate GPI/GPO

### Feedbacks:

-   Selected Source is on Selected Bus of M/E
-   Selected Source is on Selected Aux

### Variables

-   M/E: Bus: Current Source
